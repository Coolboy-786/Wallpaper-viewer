Wallpaper Viewer

A simple desktop wallpaper viewer built with Python and Tkinter. It loads images
from a local folder, displays them one at a time in a small window, and lets you
cycle through them with a Next button.

Features


Loads all images from a Wallpaper/ folder
Resizes every image to a uniform size for consistent display
Clean, black-themed window
Cycle through images with a Next button (wraps back to the start)


Requirements


Python 3.7+
Pillow (the PIL image library)
Tkinter — bundled with most Python installations
